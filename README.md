> A Vue.js 2.0 UI Toolkit for Web.

## Install
```shell
npm install gpzx-ui -save
```

## Quick Start
``` javascript
import Vue from 'vue'
import GpzxUi from 'gpzx-ui'

Vue.use(GpzxUi)

// or
import {
  Select,
  Button
  // ...
} from 'gpzx-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```

## Browser Support
Modern browsers and Internet Explorer 10+.
